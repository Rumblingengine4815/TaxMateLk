> AI tax advisory assistant for Sri Lankan freelancers and remote workers.
> Grounded in official IRD documents and the June 2026 WHT Amendment Act No. 11 of 2026.
>
> **Noob Dev — Certified AI Engineer Professional Program — Project 03 Capstone**

## Quick Start (Using uv)

```bash
# 1. Install uv if you don't have it (one-time, replaces pip+venv+pyenv)
curl -LsSf https://astral.sh/uv/install.sh | sh   # Mac/Linux
# Windows: powershell -c "irm https://astral.sh/uv/install.ps1 | iex"

# 2. Clone and enter project
cd taxmate_lk

# 3. Set up environment (uv handles Python version + venv automatically)
uv sync

# 4. Set up your API keys
cp .env.example .env
# Edit .env and add your real keys (Groq, Gemini, HuggingFace)

# 5. Test all APIs work before building
uv run python tests/test_apis.py

# 6. Drop your IRD PDFs into knowledge_base_raw/ then convert them
uv run python extract_pdfs.py

# 7. Build the vector store (ChromaDB)
uv run python src/ingest.py

# 8. Launch the app
uv run python src/app.py

# 9. Run the evaluation harness
uv run python src/compare_models.py
```

## Tech Stack

| Layer | Tool | Why |
|---|---|---|
| LLM (primary) | Groq — Llama 3.1 8B | Free, 14,400 req/day, fast |
| LLM (fallback) | Gemini 1.5 Flash | Free, 1,500 req/day, multimodal |
| Vector store | ChromaDB (local) | Zero config, runs on laptop |
| Embeddings | sentence-transformers | Local, no API key, all-MiniLM-L6-v2 |
| Agent | LangChain | Best agent + tool calling support |
| Fine-tuning | QLoRA via Groq dashboard | Free, 45 min, no GPU needed locally |
| UI | Gradio ChatInterface | Deploys to HF Spaces with one command |
| Deployment | HuggingFace Spaces | Free public URL for Gold badge |
